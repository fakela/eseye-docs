# Eseye test profile

Eseye supplies a test profile for downloading onto a v5 or v6 multi-IMSI bootstrap for testing network connectivity or equipment.

For an IoT device to register on the test equipment, the authentication parameters on the SIM must match those loaded onto the test network.

Eseye can encrypt the test profile, and load it onto a SIM at manufacture, through a SIM reader interface, or over-the-air (OTA).

We use the following test profile information:

- imsi: – 001001000000001
- op – FFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFF
- amf – 8000
- ki – 5C95978B5E89488CB7DB44381E237809
- algorithm – milenage
